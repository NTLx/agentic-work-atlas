# Mercado Libre frees developers' minds to focus on their mission with GitHub

GitHub's customer story presents Mercado Libre's use of GitHub Enterprise and GitHub Copilot across a large developer organization.

Reported headline metrics:

- **About 50% reduction** in time spent writing code with GitHub Copilot.
- **9,000+ developers** using Copilot.
- **100,000 pull requests merged per day**.
- **13,300 seats** listed for the GitHub deployment.

Mercado Libre standardized on GitHub Enterprise as its development platform. GitHub says the platform is used not only by developers but also by product managers and designers for collaboration.

GitHub describes Copilot as the "clear winner" among new platform features in Mercado Libre's recent trials. The customer story says Mercado Libre made GitHub Copilot available to the entire developer organization and that it improved satisfaction and productivity by automating repetitive or less engaging tasks.

Sebastian Barrios, SVP of Technology at Mercado Libre, is quoted as saying he spends half as much time arriving at correct code as before GitHub Copilot. The story also describes Copilot writing an entire script from a single comment in an early use.

## Workflow context

The case is not just individual autocomplete usage. Mercado Libre's engineering platform handles a high-volume workflow: developers merge and deploy about 100,000 pull requests per day, while GitHub Enterprise provides collaboration, automation, deployment, security tests, and repetitive-task automation.
