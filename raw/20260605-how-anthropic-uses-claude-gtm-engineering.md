# How one Anthropic seller rebuilt his team's workflows with Claude Code

Before joining Anthropic in 2024, Jared Sires had never written a line of code. As a GTM product manager, he used Claude Code to automate his inbox and sales workflows.

**Key Tools Built:**
*   **CLAFTS (Claude Drafts):** An application inside Gmail that uses the Claude API to draft replies to customer emails, matching Jared's voice and referencing latest documentation.
*   **Daily Brief & Daily Recap:** Skills that read his calendar, research meeting participants via web search, and draft follow-up emails from transcripts.
*   **Sales Plugin:** A package of 20+ skills and MCP connectors (Salesforce, Gong, BigQuery) used by 80% of Anthropic's sales organization.

Jared's role has shifted to "GTM Architect," where he designs products and works with senior engineers to implement them. He emphasizes that the technical barrier is dissolving, allowing non-technical staff to build strategic tools.
