# vibe coding 成瘾之后，我正在重建 AI 与生活的边界

## 编译摘要

### 1. 浓缩

- **核心结论1：vibe coding 的奖励回路与刷短视频同构，但更难觉察**
  - 关键证据：作者自陈"vibe coding 的奖励回路与刷短视频非常相似，都是一套高频、低门槛、强刺激的奖励系统"，区别在于奖励形式不是"被动消费内容"而是"主动生成结果"。输入 prompt → AI thinking → 生成代码 → 返回 demo，全程无明显阻力且每步伴随即时反馈，通过多巴胺系统强化"行为-回报"路径。
  - 关键证据：作者指出更难觉察的根因——vibe coding "披着创造、学习、提升效率的外衣"，让人觉得自己在做正经事、持续产出，难以意识到已被成瘾机制控制。刷短视频起码能意识到在"消磨时间"，vibe coding 关联的是创造欲、成就感、价值感与自我扩张想象。

- **核心结论2：高强度 AI 使用导致感知钝化与日常记忆模糊**
  - 关键证据：作者自陈三个月高强度 vibe coding 后"日常记忆开始变得模糊，常想不起昨天发生了什么"。归因于密集决策导致前额叶"决策疲劳"，进而判断力下降 + 感知能力钝化（难以觉察情绪变化、感受生活细节）；同时挤掉 offline time，短期记忆难以巩固。
  - 关键证据：觉察迟滞——直到收拾书桌才发现一个月没看书，发财树长新叶、楼下树开花都不自知，才意识到"正在一点点让渡作为人的主体性：不再由我决定注意力该投向哪里"。

- **核心结论3：定时记录（日记/周记）是让生活重新可见的硬信号**
  - 关键证据：作者靠长期写日记和每周手帐填补记忆空缺。当下觉得生活单调重复，回头翻记录才发现去了很多地方、见了朋友、发生很多事。"很多转瞬即逝的感受如果不被记录，很快被遗忘，只留下'好像什么都没发生'的模糊印象。"
  - 关键证据：定时记录的最大意义是"让我重新看见生活本身"，由此产生重建边界的动机。

### 2. 质疑

- **关于"结论1"的质疑**：作者将 vibe coding 成瘾归因于多巴胺奖励回路，但属个人体验推论，未提供神经科学实证。"前额叶决策疲劳"等说法是作者的自我解释（"我也是后面才知道"），非一手研究引用，证据强度应标注为体验性观察而非医学结论。
- **关于"结论2"的质疑**：记忆模糊与感知钝化的因果归因（AI 使用 → 决策疲劳 → 钝化）是单一案例的反推，可能混淆了"工作强度"与"AI 使用"两个变量——三个月高强度开发本身就会导致决策疲劳，未必是 vibe coding 独有的机制。但"觉察窗口被压缩"这一点有结构性意义。
- **关于数据可靠性的质疑**：本文是个人博客/公众号一手自述，n=1，无对照。其价值在于现象的精细内省而非统计代表性。"创造型成瘾比消费型更难觉察"这一判断需更多案例或研究印证。

### 3. 对标

- **跨域关联1（成瘾医学）**：vibe coding 成瘾机制与 Anna Lembke《多巴胺国度》的"高多巴胺奖励间隔缩短 → 慢反馈耐受力下降"框架吻合。作者自陈"大脑适应高强度短周期刺激后，对慢反馈、低刺激、需长期投入之事的耐受力下降"——正是 Lembke 所述多巴胺基线上调后的痛觉区间压缩。
- **跨域关联2（数字劳动批判）**：与 Byung-Chul Han《倦怠社会》的"自我剥削"同构——vibe coding 披着"创造/效率"外衣，使人主动而非被动地延长劳动，主体性让渡发生在"自觉在做正经事"的伪装下。这与 07-15 深度思考的"创造型成瘾 > 消费型成瘾"判断互证。
- **跨域关联3（东亚优绩主义）**：作者读黑塞《在轮下》引出优绩主义批判——评价体系只认成绩/结果/效率/排名，人的感受与"喜不喜欢"被抹去。作者期待"AI 抹平智力差后优绩主义崩盘，让判断、感受、选择生活的能力重新被看见"——将 AI 视为优绩主义的去阈化冲击。

### 4. 重建边界的三动作（extracted）

作者提出的个人层对抗措施（与 [[SHIELD]] 个人摩擦、07-15 增强陷阱对抗框架一致）：

1. **重新规划每天时间**：切割 vibe coding / 阅读 / 日记 / 运动，主动安排"慢下来"的事
2. **深度复盘 vibe coding 习惯**：把工作流沉淀成 skill，做"复利工程"，任务拆分并行——让 AI 在更清晰结构下发挥效率（与 [[Friction-as-Design-Signal]] 互证：刻意保留的摩擦=复利）
3. **允许自己停下来**：屏蔽焦虑噪音，承认"不必持续输入产出才算没浪费时间"

作者的红线："绝对不会在社交平台上发表 AI 生成的观点"——主体性边界的人格化表述。
