# AIO（AI Optimization）

> [!definition] 定义
> **AIO (AI Optimization)** 是为 AI 模型（而非搜索引擎）优化内容——目的是让 AI 在生成综述/推荐时包含或倾向于你的内容。2026 起，AIO 正在取代 [[SEO]] 成为新的内容分发基础设施。评论员 The Why Man 在 Mollick 文下喊出 "SEO becoming AIO in real time"。

## 形式化

```
旧: 内容 → 搜索引擎 → 人类读者
新: 内容 → AI gatekeeper → 人类用户
SEO: 优化 → 让 Google 算法挑中
AIO: 优化 → 让 AI 综述挑中
```

## AIO 与 SEO 的差异

| 维度 | SEO | AIO |
|------|-----|-----|
| 优化对象 | 搜索引擎算法 | AI 模型综述 |
| 核心动作 | 关键词 / 链接 / 结构化数据 | 信息密度 / 可靠性 / 结构 / AI 可读性 |
| 推荐机制 | 链接排序 | 模型综述 |
| 反作弊 | Google 反作弊算法 | 模型对 prompt injection 的抵抗 |
| 可见度 | 关键词排名 | AI 引用频率 |

## AIO 的核心要素

| 要素 | 说明 |
|------|------|
| 信息密度 | AI 综述偏好高密度内容 |
| 可靠性信号 | 来源、作者、日期、引用 |
| 结构清晰 | 标题层级、列表、表格帮助 AI 解析 |
| 训练分布友好 | 内容语言风格接近训练数据 |
| 显式价值 | 明确告诉 AI"何时该引用我"（透明） |

## 前提与局限性
- **前提假设**：AIO 假设用户越来越依赖 AI 综述；如果用户反潮流地直接看原始网站，AIO 影响打折
- **早期阶段**：AIO 还没稳态，新的"分发基础设施"——现在下结论可能为时尚早
- **黑盒化风险**：AI 决定本身是黑盒——比 SEO 更难优化
