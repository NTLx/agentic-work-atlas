# Chad Whitacre

> [!definition] 定义
> Chad Whitacre 是 Open Source 可持续性实践者，曾参与 Open Source Endowment 和 Open Source Pledge。2026 年 5 月，他宣布离开 tech 和 Open Source，转向离线生活，并把 AI 描述为压垮其 Open Source 动力的最后一根稻草。

## 在本库中的意义

Chad Whitacre 的价值不在于提供 AI 行业数据，而在于提供一个罕见的一手反应：一个长期参与 Open Source 可持续性的人，不是因为 AI 不能提高效率而离开，而是因为 AI 和互联网生活方式已经与他想成为的人发生冲突。

这个案例补充了本库中偏工程、组织和生产力的材料：

- 对 [[AI-Restraint]]：克制不只是系统设计，也可能是退出和拒绝。
- 对 [[Conscious-Creators]]：有意识创造可能需要离线制度，而不只是个人专注力。
- 对 Open Source 讨论：AI 可能同时增加开源生产能力和削弱部分维护者的意义感。

## 关键事件

- 2026-05-28: 发布 [[I Am Retiring from Tech to Live Offline]]。
- 2026-05-29: 计划结束在 Sentry 的工作。
- 2026-08: 计划保持在线至此，以完成 Open Source Endowment 交接。
- 之后: 转向 Home Depot 工作，并推动离线杂志 Gift。

## 关键数据点

- 作者称 AI was the last straw，但同时说明更长期的背景是 Open Source 领导角色和内在状态之间越来越难协调。
- 作者计划把 Open Source Endowment 过渡给新的 board member，并由 Vlad-Stefan Harbuz 接手 Open Source Pledge。
- 作者选择 Home Depot 工作，不是“后经济”退出，而是离开 tech 生活方式。
- 作者将自己的路线命名为 AI Amish / Internet Amish / Neo-Amish。
