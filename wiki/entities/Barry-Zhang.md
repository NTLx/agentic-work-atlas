# Barry Zhang

## 简介

Barry Zhang 是 Anthropic 的研究工程师（Research Engineer），专注于 Claude 的代理架构（agent architecture）和 Skills 系统开发。他是"Building Effective Agents"文章的合著者，在 AI 代理领域有重要贡献。

## 主要工作

- **Claude Agent Architecture** - 设计和开发 Claude 的代理系统架构
- **Skills System** - 构建 Claude 的技能系统
- **Agent Research** - 研究如何构建更有效的 AI 代理

## 核心观点

### Agent Skills

- 今天的代理"缺乏专业知识"，经常错过上下文
- 应该构建 Skills 而非 Agents - 更模块化、可组合的设计
- Skills 是代理的能力单元，应该独立开发和测试

### Effective Agents

与 Erik Schluntz 合著的"Building Effective Agents"提出：
- 简单、可组合的模式优于复杂框架
- 明确定义代理的职责边界
- 工具使用和工作流设计的重要性
