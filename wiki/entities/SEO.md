# SEO

> [!definition] 定义
> **SEO (Search Engine Optimization)** 是为搜索引擎算法（关键词、链接、结构化数据）优化内容，使网页在搜索结果中排名靠前。2026 起正被 [[AIO]] 取代。

## 关键数据点

- SEO 核心三要素：内容、链接、用户体验
- 反作弊机制：Google PageRank + Penguin + Panda + 蜂鸟算法
- 优化目标：SERP 排名 → 流量 → 转化
- 与 AIO 的核心差异：人读 vs 模型读

## 前提与局限性

- 前提：用户依赖搜索引擎作为主要信息入口
- 局限：黑盒化、过度 SEO 触发惩罚、关键词堆砌易识别
- 与 AIO 的转化路径不同：SEO → 人类点击 → AIO → AI 综述引用
