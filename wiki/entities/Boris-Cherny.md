# Boris Cherny

> [!definition] 定义
> **Boris Cherny**（鲍里斯·切尔尼）是 Anthropic 核心开发者，Claude Code 的创建者，被称为"Claude Code 之父"。经济学专业出身，完全自学编程，曾任 Meta 工程师。2024 年底至今 100% 依赖 AI 生成代码，日均处理 10-30 个 PR（最高记录 150 个/天），日常工作从 iPhone 上完成。提出 Product Overhang 概念和 Agent Loops 工作流范式。

## 职业履历

| 时间 | 事件 |
|------|------|
| 中学时期 | TI-83 Plus 计算器学习编程 |
| 大学 | UC San Diego 经济学专业，中途辍学后完成学位 |
| 约 2015-2024 | Meta（Facebook）工程师 |
| 2019 | 出版《Programming in TypeScript》（O'Reilly） |
| 2024 年底 | 加入 Anthropic，在 Anthropic Labs 孵化 Claude Code |
| 2024 年 11 月 | 停止手动写代码，100% 依赖 AI |
| 2025 年 5 月 | Opus 4 发布，Claude Code 开始指数增长 |
| 2025 年 7 月 | 被 Cursor（Anysphere）挖角，两周后被 Anthropic 请回 |
| 2026 年 5 月 | Sequoia AI Ascent 2026 发表"编程已被解决"演讲 |

## 核心思想贡献

- **"Coding is Solved"**: 对个体开发者而言，手动编码不再是必要技能
- **Product Overhang**: 模型能力已超过产品释放边界，需为下一代模型构建产品
- **Agent Loops**: Cron 驱动的自动化 Agent 循环——大规模 Agent 编排的最简方案
- **Software Democratization**: 编程将从专业技能变为通用素养（印刷术类比）
- **Cross-Disciplinary Generalist**: 未来最值钱的是跨领域通才，而非专业编程能力
