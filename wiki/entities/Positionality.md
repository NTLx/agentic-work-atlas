# Positionality（位置性）

> [!summary] 一句话
> 位置性是 Agentic Work Atlas 的统一理论——它在至少 5 次独立 deep think 追本中作为底层结构涌现，统一解释"为什么 AI 能做某些事但不能承担某些后果"。核心操作是"不可脱嵌性"：信息的生产方式能否与信息的生产者分离。

## 位置性的五个维度

位置性不是一个单维概念。它从五个独立但互补的维度构成：

| 维度 | 定义 | 不可脱嵌的对象 | 来源 |
|------|------|---------------|------|
| **本体论维度** | "拥有主观视角"的本体论事实——主观经验不可还原为客观描述 | 经验主体 | Thomas Nagel |
| **默会维度** | "从之出发"的不可被注意的基底——基底不能同时是基底和被注意的焦点 | 身体的驻扎 | Michael Polanyi |
| **社会维度** | 场域中的位置 × 惯习 × 资本——社会轨迹塑造感知和判断倾向 | 社会轨迹 | Pierre Bourdieu |
| **认识论维度** | 知识的时空特异性——关键知识只存在于当时当地的那个主体中 | 局部/瞬时的信息 | Friedrich Hayek |
| **政治维度** | 负责任的定位——客观性不是"无位置"而是承认并声明位置 | 被承认/被标记的位置 | Donna Haraway |

## 功能性 vs 存在论位置性

### 功能性位置性

可被技术缓解的不可脱嵌。给 Agent 更好的工具、更丰富的上下文、更多模态的接口——这些都是在扩展"功能性位置性"的边界。Agent 能访问更多数据、理解更多格式、生成更复杂的输出。

### 存在论位置性

逻辑上不可模拟的不可脱嵌。**Polanyi 的基底悖论**是最精确的边界：

```
基底不能同时是基底和被注意的焦点
```

人类从身体出发去注意世界——身体是基底，不在焦点中。当你把注意力转向身体（基底变成焦点），你已经切换到一个新的看不见的基底。这个结构是逻辑必然，不是技术暂时做不到。

AI 处理的一切都在焦点中——训练数据、架构参数、输出——没有任何东西"从之出发"。AI 没有不能被它自己同时注意到的基底。

### Nagel 的约束 vs 条件

| | AI | 人类 |
|---|----|------|
| 不可脱嵌的本质 | **约束** (limitation)：不能超出训练分布、不能突破架构 | **条件** (condition)：会死的身体、独一的生平、不可替代的关系 |
| 可否被技术缓解 | 是——更好的训练、更大的上下文 | 否——条件不是"限制"而是"存在方式" |
| 例子 | Agent 不能处理训练分布外的任务 | Governor 不能被技术替代承担"叫停"的后果 |

## Hayek 统一公式

位置性的核心问题可以形式化为：

```
位置性 = 信息的生产方式能否与信息的生产者分离？
```

- 能分离 → AI 可以替代（功能性位置性范畴）
- 不能分离 → 只有被定位的存在才能承担（存在论位置性范畴）

这个公式统一解释了 Atlas 四个层面的核心张力：

| 层面 | 不可分离的东西 | 为什么 AI 不能替代 |
|------|---------------|-------------------|
| **软件工程** | 代码与代码责任 | 承担责任是一个"条件"（作为有后果的存在），不是"能力"（可被训练的技能） |
| **组织系统** | 行动与现场局部知识 | 关键知识只存在于当时当地的主体中——FDE 的核心价值是提供捕获这种知识的位置 |
| **知识系统** | 判断与判断者的位置 | "正常"的定义本身就是位置性的——需要不从系统内部看的外部视角 |
| **人的核心价值** | 品味/判断力与形成路径 | 不是"技能"而是"存在方式"——被身体和生平塑造成的判断者 |

## 政治位置性（Haraway 第三类）

存在论/功能性二分法漏掉了第三个维度：**谁的位置被标记？谁的位置被伪装成"普遍"？**

AI 不是"没有位置"——它有训练数据的分布位置、设计者的价值观位置、部署者的利益位置。但它的位置被伪装成"来自无处所的凝视"（the god trick）。这是最危险的——它假装自己没有视角。

政治位置性要求：AI 系统的训练数据、设计假设、部署利益应该像人类的位置一样被标记、被承认、被质疑。

## 与已有实体的关系

位置性作为统一理论，为 Atlas 中多个独立讨论提供了统一的解释框架：

- [[Taste]]：品味不可替代性 = 品味的形成路径不可与品味者分离 → 存在论位置性的具体案例
- [[Judgment]]：判断力在边界情况中最珍贵 → 边界判断力最依赖形成路径 → 位置性最不可替代的节点
- [[Human-Governor-Agent-Operator]]：Governor 橡皮图章化 → 把人类放在需要"预测世界"而非"内省自我"的问题面前
- [[Agent-Containment]]：人的外部不可替代 → "思不可自举" → 存在论位置性在安全治理中的表现
- [[Agent-Failure-Causal-Chain]]：阻断不可自举 → 任何内部防御共享系统的位置性盲区
- [[Forward-Deployed-AI-Enablement]]：FDE 的核心价值 → 提供捕获局部知识的"位置"
- [[Context-Engineering]]：Context 是一种"功能性位置性"的工程尝试——但它只能扩展功能边界，不能触及存在边界

## 开放问题

1. 五维度如何操作化为可测量指标？
2. 技术能在多快速度上扩展功能性位置性边界而不触及存在论边界？
3. "政治位置性"是否构成独立第三类，还是功能性/存在论的再标记？
4. 位置性统一理论如何转化为 Agent 架构设计的具体原则？

## 关键数据点

- 位置性在至少 5 次独立 deep think 追本中作为底层结构涌现（意图理解/元思考/品味/判断力/Governor-Operator 关系）
- 2026-06-23 元总结：18 runs 的核心发现——"位置性"是所有 runs 的统一底层结构
- 五个维度各自对应一位经典理论家：Nagel（本体论）、Polanyi（默会）、Bourdieu（社会）、Hayek（认识论）、Haraway（政治）
- Polanyi 基底悖论：基底不能同时是基底和被注意的焦点——存在论位置性的逻辑边界
- Nagel 约束 vs 条件：AI 的不可脱嵌是"约束"（可被技术缓解），人类的不可脱嵌是"条件"（作为存在方式的不可替代性）

## 前提与局限性

- 位置性作为"统一理论"是 synthesized 判断——来自 Atlas 内部的多次圆桌和 think 追本的交叉验证，尚未经过外部独立审查
- 五个维度的区分可能存在概念重叠——"本体论维度"和"默会维度"在某些解释框架下可以合并
- 政治位置性（Haraway）是否构成独立第三类还是功能性/存在论的再标签，是开放问题
- 该理论尚未被操作化为可测量的指标——目前停留在概念框架层面
- 历史上存在位置性被过度强调的风险——可能将"必须在现场"变成一种新的人机分工意识形态，阻碍 AI 可以合法替代的功能
