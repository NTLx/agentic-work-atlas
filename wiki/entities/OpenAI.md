# OpenAI

> [!definition] 定义
> **OpenAI** 是由 Sam Altman、Greg Brockman、Ilya Sutskever 等人于 2015 年创立的前沿 AI 实验室，ChatGPT 与 GPT 系列模型开发方。2026-06-04 发布 Dreaming V3 记忆系统，把 ChatGPT 的"会话"概念改写为"持续委托"——单次对话不再有清晰开始/结束，而是接续在被持续维护的"用户认知状态"上。

## 关键数据点
| 指标 | 状态 |
|------|------|
| 创立 | 2015 |
| 主要产品 | ChatGPT、GPT 系列、DALL-E、Sora、Codex |
| 2026-06 记忆系统 | Dreaming V3 上线（独立可扩展） |
| 用户规模 | 2026 年亿级 |
| 商业模式 | 订阅（Plus/Pro/Enterprise）+ API |

## Dreaming 演化时间表

| 时间 | 阶段 | 特征 |
|------|------|------|
| 2024-04 | Saved memories | 用户显式"记住 X" |
| 2025-04 | Dreaming V0 | 后台从聊天历史整理 |
| 2025-06 | Memory improvements | 短期连续性 rollout 到 free users |
| 2026-06 | Dreaming V3 | 独立可扩展系统 |

## 标志性动作：把"会话"概念改写

OpenAI 2026-06 的 Dreaming V3 真正宣告的不是"ChatGPT 有了更好的记忆"，而是"个人 AI 助手的产品架构基本定型"——后台持续综合 + 用户可审可改 + 按需注入 prompt。这三件事一旦稳态，所有 LLM 应用都会被拉向"持续助理"形态。
