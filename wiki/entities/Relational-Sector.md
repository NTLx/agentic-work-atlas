# Relational Sector（关系型部门）

> [!definition] 定义
> **关系型部门 (Relational Sector)** 是消费者为"人类在场"额外付费的服务部门——价值不来自任务完成度，而来自偏好。AGI 抬高 f(完成度)，不影响关系溢价。属于此类的工作：律师、医生、教师、心理咨询师、餐厅、奢侈品、宗教、社交陪伴。

## 形式化

```
任务价值 = f(完成度)
关系价值 = f(完成度) + 人类溢价
AGI 抬高 f(完成度) → 任务价值 ↑，关系价值 ↑ 不变
```

## 关键特征

- **溢价是市场现象不是技术现象** — 消费者主动选择"对面坐的是人"
- **监管可放大或缩小** — 医疗签字、远程手术许可直接决定关系溢价能否被替代
- **AI 是工具不是替代** — AI 完成"任务环"，人保留"关系环"
- **高端市场更明显** — 餐厅、奢侈品、心理咨询的客户几乎都愿意为"人味"付费

## 边界

- **替代品风险** — 关系型体验（陪伴、亲密）的"AI 替代"对部分用户是"够用替代"
- **监管依赖** — 当远程法律签字、远程手术被允许，溢价瞬间消失
- **国别差异** — 高收入国家/高净值消费者更愿付溢价；中低收入市场可能不同
