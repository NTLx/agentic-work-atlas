# Biological Data Infrastructure（生物数据基础设施）

> [!definition] 定义
> **Biological Data Infrastructure** 是生命科学研究的数字基座。Laura Luebbert (2026) 将其类比为“为行人设计的古老城市”，而软件基础设施则是“为汽车（Agent）设计的新型城市”。

## 关键数据点
- 现状: 大量关键数据被锁定在需要手动点击、下拉选择的 Web 界面后。
- 点击税 (The Click Tax): 科学家每天耗费大量时间在界面跳转中，而这些逻辑对机器极度不友好。

## 前提与局限性
- **前提**: 必须存在可编程接口或能够通过封装实现的确定性访问路径。
- **局限**: 异构性极高，不同的实验室、数据库（NCBI, Pathoplexus 等）遵循完全不同的 ID 体系和元数据标准。
