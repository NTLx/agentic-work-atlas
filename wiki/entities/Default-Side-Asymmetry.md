# Default-Side-Asymmetry（默认侧不对称 / 时间之矢治理定理）

> [!definition] 定义
> 任何权限制度都必然有"默认侧"——真空态（无人做决定时）权限请求要么批准要么拒绝，逻辑排中必选一侧。默认批准（持久授予）则撤销难，默认拒绝（自动过期）则续期难，两侧永远不对称。棘轮不可消除只能转移。不可消除根 = **时间之矢**：治理决定落在时间轴上，授予面向开放未来、撤销面向封闭过去，未来与过去物理上不平等。消除棘轮 = 消除时间之矢 = 不可能。

## 七层下钻：从默认到时间之矢

1. **默认**：真空态权限请求必须有唯一答案，否则系统无法运行。持久制选"默认批准"，过期制选"默认拒绝"。
2. **方向**：默认批准把"维持"成本设零、"改变"（撤销）成本设正；默认拒绝反过来。两边都把一类动作设零、另一类设正——不对称源头。不能两边都零成本（真空态同时批准又不批准=矛盾）。
3. **不可观测**：正成本分配的不只是执行成本，是更前端的触发认知成本——休眠权限无信号，主动扫描无终点，触发永不启动。
4. **判定者**：把触发交给机器？机器判据是人写的，判据即对"需要"的定义，定义即默认。换机器判据只换默认。
5. **定义即默认**：任何把世界分成"保留/失效"的操作都预设分类判据，判据即定义即默认——逻辑结构，绕不开。
6. **离散**：存废为何二元？因"调用权限"本身二元（一次调用要么成功要么失败）。治理离散性继承自被治理对象的离散性，而动作离散性是逻辑与物理底层。
7. **时间之矢**（终点）：离散对称（有/无等价），不对称从时间进入。授予=创造未来可用可能性（面向开放未来），撤销=关闭已被使用可能性（面向封闭过去）。未来开放、过去封闭=时间之矢，物理不可逆。默认必选一侧，因时间有方向、两侧物理上不平等。

## 对称反例为何不存在

Coase 提出"全权限自动过期系统"作为对称反例。它不消除棘轮，只转移：

| 续期方式 | 后果 |
|---------|------|
| agent 自续 | 续期成 agent 既得利益→零成本默认→新棘轮在续期侧重生 |
| 人审续期 | 决策成本随权限数量爆炸→高频场景不可行 |

自动过期用"高频决策成本"换"低频撤销成本"，棘轮换了边没消失。**任何制度必有默认，两侧永不对称**。

## 连续/概率权限也绕不开

放弃二元存废、用连续使用频率映射存废概率？映射函数（"频率>阈值则保留"）本身是判据，暗含"需要=有可观测使用"的定义。低频但关键的权限（一年用一次但那次至关重要）被映射为低概率、易失效。棘轮从"撤销难"变成"低频关键权限易丢失"——换形不消。

```
二元分类 → 判据 → 定义 → 默认（绕不开的逻辑结构）
连续映射 ≠ 无判据；映射函数 = 隐含定义 = 新默认
```

## 与 07-14 有穷者治理悖论的闭合

有穷者治理悖论（07-14）：开放系统中不存在有限最优治理阈值，治理=分配错误代价的权力。

本定理给出该悖论的**最深形式**：开放系统中不存在对称治理，非因有穷者能力不足，是因**对称在时间有向世界里根本不存在**。这是存在论层面的不对称，治理只是它的影子。

```
有穷者治理悖论（空间/可能性维度）：阈值不可优化
        ≅
默认侧不对称（时间维度）：阈值不可回退
        ↓
统一根：时间之矢——治理状态在可能性空间与时间轴上都单向不可逆
```

## 治理真实目标定理

既然棘轮不可消除只能转移，真实目标不是"消除棘轮"而是"把棘轮压在代价较小的时间侧"：

| 选择 | 代价 | 特征 |
|------|------|------|
| 压在未来侧（自动过期，续期难） | 续期决策成本 | 可分散到每次续期、多人分担、可自动化辅助 |
| 压在过去侧（持久授予，撤销难） | 拆除已积累依赖 | 集中爆发、连锁中断、难分担 |

通常选前者——未来不确定性可分散承担，过去依赖拆除集中爆发。但这是**选较轻的不对称，非消除不对称**。

[[Plan-is-the-Permission]] 是这个选择的局部实现：低频高价值操作用自动过期（续期难但可承受），高频操作仍用持久授予（续期成本爆炸，退而求次）。

## 关键数据点

- Vercel Agent 的 Plan-is-the-Permission 是"压在未来侧"的工程实证，但仅适用低频高价值操作（[[20260708-vercel-agent]]）
- 企业权限"只增不减"现象普遍，是"压在过去侧"的默认选择（因人审续期成本爆炸，退而求次接受撤销棘轮）
- 软件依赖"只增不断"（Brooks 概念完整性愈合）是同一时间之矢在代码层的投射

## 前提与局限性

- 本定理是 think 七层下钻的 synthesized 判断（逻辑排中 + 分类判据 + 动作离散性 + 时间之矢四层推导），未经独立形式化证明。
- "时间之矢是治理不对称的终极根"是强主张，类比热力学第二定律投射到治理域，类比强度待验证——治理的"时间"是否物理时间，还是可构造无时序的封闭治理系统？
- 证伪"棘轮不可削弱"易（Plan-is-the-Permission 已局部削弱）；证伪"棘轮不可消除"需找到开放系统的完全对称治理，预判不可能（等价于消除时间之矢）。
- 续期决策成本爆炸的阈值（权限数量超过多少则自动过期比持久更贵）未量化，是 Plan-is-the-Permission 适用边界的开放问题。
