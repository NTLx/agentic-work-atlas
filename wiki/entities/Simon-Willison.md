## 核心观点

### Agentic Engineering

Simon Willison 提出 "Agentic Engineering" 概念，定义为使用 coding agents 辅助开发软件的实践。核心洞察：

1. **Code execution is the defining capability** - 代码执行是决定性能力
2. **Writing code is cheap now** - 写代码成本已大幅下降，但好代码依然有成本
3. **Hoard things you know how to do** - 囤积你知道如何做的事情
4. **AI should help produce better code** - AI 应帮助我们产出更好的代码

### 最佳实践

- 不要将未经审查的代码提交给协作者
- 建立可运行代码示例的收集
- 使用异步编码助手处理重构任务
- 持续改进指令和质量（Compound Engineering）
