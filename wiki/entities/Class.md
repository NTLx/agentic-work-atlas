# Class (Ontology)

> [!definition] 定义
> 本体论术语：TBox 的核心元素，表示概念类型。

## 关键数据点

- **定义**: 概念类型，是 Individual 的模板
- **示例**: 'Person', 'Organization', 'Project' 都是 Class
- **位置**: 在 TBox（概念层）中定义

## 前提与局限性

- **前提**: Class 定义了概念的本质属性和关系
- **局限性**: Class 是抽象定义，不包含具体数据
