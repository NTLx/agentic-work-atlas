# Cat Wu

> [!definition] 定义
> **Cat Wu**（Catherine Wu）是 Anthropic Claude Code 和 Cowork 的产品负责人。
> 她毕业于普林斯顿大学计算机科学专业，曾任 Scale AI 产品工程师、Dagster 工程经理、Index Ventures 风投投资人。
> 2024 年 8 月加入 Anthropic 后，她与 Boris Cherny 搭档推进 Claude Code 产品线，并持续观察 AI 时代 PM 能力结构的变化。

## 核心思想贡献

- **AI-Native Shipping**: 产品发布周期从月到天——不是靠更好模型，而是靠流程和文化
- **Research Preview 策略**: 降低发布门槛的核心方法论
- **PM 角色转型**: PM 从 roadmap 对齐者变为"高速公路设计师"
- **Model Introspection**: 最 underrated 的 AI 技能——让模型自省错误
- **Mission Alignment 为组织效率**: Anthropic 极速迭代的底层驱动
- **Delete Features > Add Features**: 新模型让删除功能比新增功能更有价值
