# Tobi Lütke

> [!definition] 定义
> **Tobi Lütke** 是 Shopify 的创始人兼 CEO。他 16 岁从德国学校辍学进入 Siemens 子公司担任学徒程序员（Apprentice Programmer），这一经历深刻影响了他后来对软件工程和组织发展的看法。

## 可验证履历

| 时间 | 事件 |
|------|------|
| ~1998 | 16 岁辍学，进入 Siemens 子公司担任学徒程序员 |
| 2006 | 创立 Shopify（最初是在线滑雪板商店 Snowdevil 的技术转型） |
| 2013 | 发表 *The Apprentice Programmer*，主张学徒制优于课堂教育 |
| 2018 | 发表 *The Future Role of Human Excellence*，讨论人类卓越在自动化时代的角色 |
| 2026 | 推动 Shopify 内部 River Agent 和 Lehrwerkstatt 模式 |

## 核心观点

### 学徒制与渗透式学习

Lütke 认为程序员的最佳培养方式不是课堂，而是”坐在高手旁边看他们如何做决定（[[Judgment]]）”。这一观点直接源自他的学徒经历——在 Siemens 子公司，他通过观察资深程序员的决策过程学到了比课堂更多的东西。

在 AI 时代，他将这一理念扩展为 [[Lehrwerkstatt|教学工坊]]：将整个公司变成一个教学工坊，通过公开 AI 交互实现 [[Osmosis-Learning|渗透式学习]]。Shopify 的 River Agent 只在公开频道工作、不接受 DM 的 [[Public-only-Constraint|公开频道约束]]，正是这一理念的制度化表达。

### 组织速度与透明度

Lütke 认为组织的速度受限于”最慢秘密”的流通速度。他主张极度透明以减少沟通带宽的损耗——私聊、会议和邮件会把知识锁在个人或小组内部，而公开 Agent 对话则可搜索、可复写、可引用。

### AI 时代的组织设计

Lütke 在 2026 年推动 Shopify 的 River Agent 成为组织学习基础设施。River 过去 30 天被 5,938 名员工在 4,450 个频道中使用或观察，PR merge rate 在两个月内从 36% 提升到 77%——不是因为底层模型切换，而是因为员工持续改进 River 的 instructions、memory 与 skills。
