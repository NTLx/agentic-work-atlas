> [!definition] 定义
> Kevin Xu 是 GitHub CELA（Corporate, External, and Legal Affairs）团队的 Staff Software Engineer，GitHub Innovation Graph 项目的核心推动者。他通过系列博客文章将学术研究者与 GitHub 数据连接起来，促进开源软件经济影响的研究。

## 背景

- GitHub 用户名：@khxu
- 贡献项目：github/innovationgraph、github/government-open-source-policies、observablehq/plot、openai/GABRIEL 等
- 合著论文：Harvard Business School "Generative AI and the Nature of Work"（2024），与 MIT、Harvard 研究者合作
- 博客系列：持续发布 Innovation Graph 季度数据更新和研究者访谈

## Innovation Graph 博客系列

Kevin Xu 主导的系列访谈，将 GitHub 数据开放给经济学研究者：
- 软件经济复杂度研究（本文）
- ChatGPT 对软件开发活动的影响
- GitHub Copilot 对开源维护者工作方式的因果效应
- 季度数据可视化与趋势分析
