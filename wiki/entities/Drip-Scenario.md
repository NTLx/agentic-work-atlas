# Drip Scenario（渐进式失业场景）

> [!definition] 定义
> **Drip Scenario** 是工人被持续低烈度推到更低薪岗位的渐进式失业模式。Imas 在 Dwarkesh 对谈中警告：drip scenario 比一次性大冲击更政治危险——它累积痛苦可能 ≥ 冲击，但没有"事件"的政治可见度，难以触发政策响应。

## 形式化

```
旧: 失业 = 一次性事件
新: 失业 = drip（持续低烈度）+ 偶尔 shock（高烈度）
drip 累积痛苦 ≥ shock ∧ drip 政治可见度 < shock
```

## 与大冲击对比

| 维度 | Drip Scenario | 大冲击 Shock |
|------|---------------|---------------|
| 烈度 | 低 | 高 |
| 持续时间 | 长（5-20 年） | 短（< 1 任期） |
| 政治可见度 | 低（"正常调整"） | 高（"危机"） |
| 政策响应 | 弱 | 强 |
| 累积痛苦 | 隐性高 | 显性高 |
| 政策可解性 | 难（缓慢，难命名） | 易（事件触发干预） |

## 现实观察

- 当下劳动力市场（除初级开发）= 略低于趋势，不是"血洗"
- 1980s 制造业外迁 = drip 案例，被 GDP 增长吸收
- 北欧国家几乎用社保吸收 drip；美国无此缓冲
- 初级开发招聘"略低于趋势"是 drip 早期信号

## 前提与局限性
- **政治学论述而非经济学预测** — 概率取决于工种替代速度 vs 新岗位诞生速度差值
- **国别差异大** — 制度吸收能力决定 drip 痛苦度
- **GDP 增长可吸收 drip** — 1980s 美国如此；2026 之后是否仍如此未知
